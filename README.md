# Semora — CGPA Calculator

A clean, earthy-toned CGPA calculator built with React 19, Vite, and Tailwind CSS. Track grades across semesters, visualize trends, and set academic goals — all stored locally in your browser.

## Features

- **Preset Curriculum** — Pre-loaded courses for SKCET B.Tech IT (2024–2028, Regulation 2022)
- **Real-time SGPA/CGPA** — Instant calculation as grades are entered
- **Multi-profile** — Separate profiles with 4-digit PIN protection
- **Semester Toggle** — Include/exclude semesters from CGPA calculation
- **Grade Scale** — Customizable grade-to-point mapping
- **Trends Chart** — SGPA/CGPA progression over semesters (Recharts)
- **Grade Distribution** — Donut chart breakdown of achieved grades
- **Target CGPA** — Set a goal and see required future GPA
- **Dark/Light Mode** — Toggle with system preference support
- **Responsive** — Mobile, tablet, and desktop layouts
- **Data Persistence** — Everything saved to localStorage

## Tech Stack

- React 19 + Vite 6
- Tailwind CSS 4
- Recharts 3.9
- Lucide React (icons)
- React Router DOM 7
